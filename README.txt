RECONECTA Static Site (v1.0)

Incluye:
- Sitio multi-página HTML/CSS/JS.
- CTA “Visor/Atlas” en el menú (abre en pestaña nueva).
- Página visor.html con iframe responsive incrustando: https://reconecta.shinyapps.io/visor/

Despliegue rápido (GitHub Pages):
1) Crea un repo en GitHub (p.ej. reconecta-site).
2) Sube el contenido de este ZIP a la raíz del repo.
3) Settings → Pages → Deploy from a branch → Branch: main / root.
4) La URL quedará como https://<usuario>.github.io/<repo>/

Cloudflare Pages / Netlify:
- Publica como sitio estático (sin build). Carpeta raíz.

Edición:
- Cambia textos en los .html.
- Logotipos: coloca imágenes en assets/img/logos/ y enlázalas en el footer si lo deseas.
